# Windchill 零部件搜索与浏览操作指南

> 适用版本：Windchill PDMLink 11.x / 12.x
> 文档版本：v1.0

---

## 前置条件

- 拥有 Windchill 登录账号
- 了解需要搜索的零部件信息（编号/名称/类型）

---

## 操作步骤

### 步骤 1：进入搜索界面

**方式一：快速搜索**
1. 在 Windchill 主界面顶部导航栏找到 **搜索框**
2. 输入关键词（物料编号、名称等）
3. 按回车或点击搜索图标

**方式二：高级搜索**
1. 点击搜索框旁的 **"高级搜索"（Advanced Search）**
2. 打开多条件搜索面板

### 步骤 2：设置搜索条件

| 条件字段 | 说明 | 示例 |
|---------|------|------|
| 编号（Number） | 精确搜索物料编号 | MB-001 |
| 名称（Name） | 模糊搜索物料名称 | 螺栓 |
| 类型（Type） | 按物料类型筛选 | 标准件/自制件 |
| 状态（State） | 按生命周期状态 | 已发布/在制 |
| 创建者（Created By） | 按创建人 | 用户名 |
| 创建日期（Created Date） | 按时间段 | 最近30天 |

### 步骤 3：执行搜索

1. 填写搜索条件后，点击 **"搜索"（Search）**
2. 系统显示搜索结果列表
3. 可以使用 **"保存搜索"（Save Search）** 保存常用搜索条件

### 步骤 4：浏览搜索结果

1. 结果列表默认显示：编号、名称、版本、状态、类型
2. 点击列标题可排序
3. 点击 **"自定义列表"** 可增删显示列
4. 点击物料名称进入 **详细信息页**

### 步骤 5：查看零部件详情

在详细信息页可以查看：
- **基本信息**：编号、名称、版本、状态、类型
- **属性**：材质、重量、单位等扩展属性
- **结构（Structure）**：BOM 结构树
- **关联文档**：图纸、工艺文件等
- **历史版本**：版本变更记录

## 搜索技巧

| 技巧 | 说明 |
|------|------|
| 通配符搜索 | 使用 `*` 或 `%` 模糊匹配，如 `M8*` |
| 精确搜索 | 加引号搜索精确短语，如 `"M8×30螺栓"` |
| 最近浏览 | 在首页 **"最近浏览"** 中查看已看过的物料 |
| 收藏夹 | 常用物料可添加到 **"收藏夹"（Favorites）** |

## 常见问题

### Q1：搜不到想要的物料？

尝试简化关键词、使用通配符、或联系文档管理员确认物料是否存在。

### Q2：怎么查看物料的所有版本？

在物料详情页 → **"版本历史"（Version History）** 标签页查看。
