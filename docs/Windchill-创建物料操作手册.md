# Windchill 创建物料（部件）操作手册

> 适用版本：Windchill PDMLink 11.x / 12.x
> 文档版本：v1.0

---

## 前置条件

- 拥有 Windchill 系统登录账号
- 拥有"部件创建"权限（WTUser 角色以上）
- 了解企业物料编码规则

---

## 操作步骤

### 步骤 1：登录 Windchill 系统

1. 打开浏览器，访问 Windchill 系统地址（如 `http://windchill.company.com/Windchill`）
2. 输入用户名和密码，点击"登录"
3. 进入主界面后，确认左上角显示的产品库（Product）正确

### 步骤 2：进入部件管理模块

1. 在导航栏中点击 **"部件"（Parts）** 标签
2. 在左侧文件夹树中，选择目标文件夹（如 `/产品库/通用件` 或 `/项目/XXX项目`）
3. 右侧列表显示该文件夹下已有的部件列表

### 步骤 3：新建部件

1. 点击列表上方的 **"新建" → "部件"（New → Part）** 按钮
2. 在弹出的新建对话框中填写以下字段：

| 字段 | 必填 | 说明 |
|------|------|------|
| 编号（Number） | 是 | 按企业编码规则填写，或留空系统自动生成 |
| 名称（Name） | 是 | 填写物料标准名称，如 "M8×30螺栓" |
| 类型（Type） | 是 | 选择：自制件/外购件/标准件 等 |
| 单位（Unit） | 是 | 默认 "件"（EA） |
| 版本（Version） | 否 | 默认自动生成 |

### 步骤 4：填写部件属性

1. 点击 **"确定"** 后，进入部件详情编辑页面
2. 切换到 **"详细信息"（Details）** 标签页
3. 填写以下属性（按企业规范）：

**基础信息：**
- 默认单位：EA（件）
- 来源：自制/外购
- 状态：在制/已发布

**组织信息：**
- 创建者：自动获取当前登录用户
- 组织：按实际选择

**分类属性（如有配置）：**
- 物料组：按分类选择
- 重量：填写净重
- 材料：填写材料牌号

### 步骤 5：保存并完成

1. 点击 **"完成"（Finish）** 按钮
2. 系统提示"部件创建成功"
3. 系统自动生成物料编号（如未手动填写）
4. 新部件出现在部件列表中

## 常见问题

### Q1：无法创建部件，按钮灰色？

可能原因：
- 当前用户没有部件创建权限 → 联系管理员赋权
- 当前文件夹不允许创建 → 切换有权限的文件夹
- 产品库处于只读状态 → 联系管理员

### Q2：物料编号能自动生成吗？

取决于系统配置。如果开启了自动编码（Auto Numbering），留空编号字段即可自动生成。如果需要手动编码，请参考《物料编码规则》。

### Q3：创建后如何修改？

进入部件详情页 → 点击 **"编辑"（Edit）** → 修改属性 → **"保存"（Save）**。注意：已发布的部件部分属性不可修改。

### Q4：审批流程是什么？

新创建的部件默认状态为"在制"。如需发布，需发起 **"变更请求"（Change Request）**，经过审批流程后状态变为"已发布"。
